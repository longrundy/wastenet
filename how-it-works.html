The method behind the madness

Every number on both dashboards is worked out by a rule. This page is where the rules are written down — in plain terms, not in code.

It is the same text you get from the small i beside any figure on either dashboard. Not a copy of it: the same text, from the same file. There is only one explanation, so it cannot say one thing here and something else there.

If anything on this page does not match what the dashboard actually does, the page is wrong — and it should be reported, not worked around.

One thing worth understanding before anything else.
You will notice this page almost never gives you a number. It does not say “we flag a box at 80% full”. It says we flag a box when it reaches its own trigger, and then the dashboard shows you what that trigger actually is, today, for that box.

That is deliberate. A number written down here becomes a lie the day somebody changes it in the engine — and it lies with the authority of being printed inside the tool, which a new person will believe over the code. A number that was never written down cannot go stale.
